### 📁 Featured Projects

#### 🔒 [Federated Biometrics](https://github.com/jaismathew-13/federated-biometrics)
*Finger vein authentication using Siamese-EfficientNet + Federated Averaging on MMCBNU_6000.*
- **Core Architecture:** Trained a Siamese network with an EfficientNet backbone to achieve high-accuracy biometric verification.
- **Privacy First:** Implemented Federated Averaging, allowing edge nodes to train locally and share weights rather than raw biometric data.
- **Stack:** `Python`, `TensorFlow`, `Siamese Networks`, `Federated Learning`

#### 🍯 [HoneyTrap System](https://github.com/jaismathew-13/honeytrap)
*Flask-based bot detection with browser fingerprinting, behavioral analysis, and admin dashboard.*
- **Detection Engine:** Implemented active and passive client profiling (canvas, WebGL, user-agent) paired with behavioral mouse-tracking.
- **Admin Hub:** Features a real-time web dashboard displaying live traffic logs, threat levels, and IP blocklisting controls.
- **Stack:** `Flask`, `Python`, `SQLite`, `JavaScript`, `CSS3`

#### 🕳️ [PotholeGuard](https://github.com/jaismathew-13/potholeguard)
*Real-time road defect detection using YOLOv12 + OpenCV.*
- **Computer Vision:** Fine-tuned YOLOv12 to perform rapid anomaly detection on road pavement from dashcam footage.
- **Real-Time Pipeline:** Built frame-by-frame processing using OpenCV to overlay bounding boxes with minimal latency.
- **Stack:** `YOLOv12`, `OpenCV`, `Python`, `PyTorch`

#### 📧 [Gmail Classifier](https://github.com/jaismathew-13/gmail-classifier)
*Chrome extension with local AI classification via Ollama (llama3.2).*
- **Private local AI:** Parses incoming email contents and dynamically categorizes them locally to guarantee message privacy.
- **DOM Integration:** Built a robust Chrome content script that injects category badges directly into the Gmail web interface.
- **Stack:** `JavaScript`, `Ollama (Llama 3.2)`, `Chrome Extension API`, `HTML5`, `CSS3`
