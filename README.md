### 📁 Featured Projects

#### 🤖 [JobDecode AI](https://github.com/jaismathew-13)
*FastAPI + Gradio RAG application using FAISS and Groq's Llama 3.3 70B for resume and job description analysis.*
- **Parsing & Mapping:** Parses complex PDF templates and maps candidate experiences directly to specific tech-industry roles.
- **Retrieval-Augmented Generation:** Uses FAISS vector databases to enable contextual, high-accuracy Q&A over resume data.
- **Stack:** `FastAPI`, `Gradio`, `FAISS`, `Llama 3.3 (Groq)`, `RAG`, `Python`

#### 🛡️ [Federated Biometrics](https://github.com/jaismathew-13/federated-biometrics)
*Finger vein authentication research using custom EfficientNet + DiagonalAdapter trained via Federated Averaging.*
- **Biometric Authentication:** Trained a Siamese network with an EfficientNet backbone to achieve high-accuracy biometric verification.
- **Privacy-Preserving ML:** Implemented FedAvg across distributed clients on the MMCBNU_6000 dataset (~1,280 trainable params) for edge-only training.
- **Publication:** IEEE SPICES 2026 conference submission comparing six deep learning architectures.
- **Stack:** `Python`, `TensorFlow`, `Siamese Networks`, `Federated Learning (FedAvg)`

#### 🍯 [Adaptive Bot Detection & HoneyTrap](https://github.com/jaismathew-13/honeytrap)
*Browser fingerprinting and behavioral mouse-tracking system with decoy honeytrap redirection.*
- **Detection Engine:** Analyzes visitor risk based on active/passive canvas, WebGL, user-agent profiling, and mouse dynamics.
- **Attacker Simulation:** Tested and validated against an adversarial bot built with the Gemini Vision API (gemini-2.0-flash) performing CAPTCHA-solving.
- **Stack:** `Flask`, `Python`, `Browser Fingerprinting`, `Gemini Vision API`, `JavaScript`, `CSS3`

#### 🕳️ [PotholeGuard](https://github.com/jaismathew-13/potholeguard)
*Real-time road defect detection system using YOLOv12 + OpenCV.*
- **Computer Vision:** Fine-tuned YOLOv12 to detect pavement anomalies on live video feeds with high confidence scores.
- **Low-Latency Pipeline:** Integrated frame-by-frame preprocessing via OpenCV for real-time bounding box visualization.
- **Stack:** `YOLOv12`, `OpenCV`, `PyTorch`, `Python`

#### 🚗 [Used Car Price Predictor](https://github.com/jaismathew-13)
*Log-Log Ridge Regression model with glassmorphic analysis dashboard.*
- **ML Engine:** Achieved 96.0% validation R² score and halved mean absolute error to 0.60 Lakhs using Ridge Regression.
- **Diagnostics Dashboard:** Displays real-time feature coefficients, residual analysis, and interactive diagnostic scatter plots.
- **Stack:** `Flask`, `scikit-learn`, `Python`, `Chart.js`, `HTML5/CSS3`
